# Yarn

- `yarn init`：初始化，创建 `package.json`
- `yarn add 包名`：添加依赖
- `yarn add 包名 -D`：添加开发依赖
- `yarn remove 包名`：移除包
- `yarn`：自动安装依赖
- `yarn run`：执行自定义脚本
- `yarn <指令>`：执行自定义脚本
- `yarn global add`：全局安装
- `yarn global remove`：全局移除
- `yarn global bin`：全局安装目录

# pnpM

- `pnpm init`：初始化项目，添加 `package.json`
- `pnpm add 包名`：添加依赖
- `pnpm add -D 包名`：添加开发依赖
- `pnpm add -g 包名`：添加全局包
- `pnpm install`：安装依赖
- `pnpm remove 包名`：移除包
